Cortex-Plattform

Offenes Framework für Arztpraxen — Distribution + Doku auf einen Blick.

Erst-Installation

curl -fsSL https://sanexio.github.io/cortex/install.sh | bash

Lädt das signierte und Apple-notarisierte Cortex-CLI für dein Mac-Target (arm64 oder x64), legt den Befehl cortex in ~/bin/ an, prüft SHA-256.

Vollständige Anleitung

Das Onboarding-Tutorial zeigt dich Schritt für Schritt: Voraussetzungen, Lizenz, Wizard, Tenant-Inhalte, Site-Bau, DSGVO, häufige Fehler, FAQ.

Endpoints

  • install.sh — Bootstrap-Skript
  • api/v1/update/manifest.json — Cortex-CLI Update-Manifest
  • dist/ — signierte Cortex-CLI-Bundles (arm64 + x64)
  • nexus/api/v1/update/manifest.json — Nexus-Gateway Update-Manifest
  • nexus/dist/ — signierte Nexus-Bundles (arm64 + x64)
  • docs/ — Onboarding-Tutorial

Repositories

  • Sanexio/cortex-cli — Kommandozeilen-Werkzeug
  • Sanexio/cortex-web — Framework (Theme, Schemata, Adapter)
  • Sanexio/cortex-tenant-template — Vorlage für neue Tenants
  • Sanexio/nexus — Plattform-Spec und Roadmap

Gehört zur Sanexio-Organisation. Lizenz: Apache 2.0 für Framework und Werkzeuge. Tenant-Inhalte gehören dem jeweiligen Arzt-Tenant.
